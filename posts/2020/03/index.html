Announcing {spacey}, now on CRAN!

USGS data access and rayshader maps, done cheap.
R
R Packages
maps
spacey
geospatial data
Author

Mike Mahoney

Published

March 24, 2020

I’ve launched a new package to CRAN! spacey helps you pull elevation and image overlay data from the USGS and ESRI, then helps you turn them into beautiful maps via the fantastic rayshader package.

The package has a documentation website built with pkgdown – check it out for more information!
